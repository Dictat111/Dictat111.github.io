hexo命令的回顾

Dictat
2023-10-02
hexo, 网站建立

“故地重游,月圆更寂寞~ ~”
由于已经有一年多没有使用hexo了, 我对个这工具的大多数命令已经十分的生疏,有必要复习一下相关的知识了.

生成静态文件和部署

hexo generate生成
hexo deploy部署

本地测试

hexo server 在本地生成一个服务器

新建文章？

比如用
hexo new "hexo命令的回顾"

关于标签的使用

在使用多个标签的时候,需要用使用方括号,格式如下:
tags: [数学,拓扑学,数学问题]

插入图片？

由于之前的设定,我在typora之中插入图片的同时,会自动在当前目录下生成一个文件夹并且将图片复制到其中.
