发表于 2024-03-24 更新于 2024-03-25 Disqus：

如何更改主题颜色

快捷键

移动行

ctrl + shift + enter 光标跳转到上行

ctrl + enter 光标跳转到下行

复制这一行到下方

ctrl + shift + ↓

多选

按住 alt 键然后用鼠标点击,能使得光标出现在多个位置

按住 ctrl + alt 加方向键上下,把光标复制到上下多行.

ctrl + shift + L ,把光表移动到所有与选中的文本相同的文本后.

CTRL + D 批量更改变量（多按几次,和上面那个类似,但是是逐次增加的）

CTRL + L 选择整行

CTRL + LEFT/RIGHT 光标跳到下一个单词

Ctrl + j 隐藏底部控制面板 ??

CTRL + C 复制当前行
CTRL + X 删除当前行
CTRL + SHIFT + F全局搜索
CTRL + G 跳转到指定行

删除当前行：shift + ctrl + k

控制台终端显示与隐藏：ctrl + ~

查找文件/安装vs code 插件地址：ctrl + p

代码格式化：shift + alt +f

新建一个窗口: ctrl + shift + n

行增加缩进: ctrl + [

行减少缩进: ctrl + ]

裁剪尾随空格 Ctrl + K 然后 Ctrl + X (支持多行)

自动换行: alt + z

快速回到顶部 ： ctrl + home

格式化选定代码 ：ctrl + k / ctrl +f

展开代码： ctrl + k + j (完全展开代码)
